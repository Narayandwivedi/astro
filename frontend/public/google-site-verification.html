Google Site Verification for Astro Satya

This page is for Google Search Console verification.

To complete setup:

  1. Go to Google Search Console
  2. Add your property (astrosatya.com)
  3. Choose HTML file verification method
  4. Download the verification file
  5. Replace this file with your verification file
  6. Upload to your server's public folder
  7. Click "Verify" in Google Search Console
